Request transformation happens before the actual request will be made. At this point the input parameters will be chained, separated by colon and a hashed output will be created. This will be the actual data to be sent by the request. This data will be then regenerated at the server knowing some of the inputs upfront.


HmacSHA512 algorythm is responsible for the high level data integrity. The output then will be encoded.
